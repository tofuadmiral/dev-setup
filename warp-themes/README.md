# Warp Themes

## Gruvbox Dark Hard (High Contrast)

A higher-contrast variant of Gruvbox Dark for Warp terminal.

### Install

```bash
mkdir -p ~/.warp/themes
cp gruvbox-dark-hard.yaml ~/.warp/themes/
```

Then in Warp: `Cmd+P` → "Open Theme Picker" → select "Gruvbox Dark Hard".

### Matching Slack sidebar

Settings → Themes → Custom Theme, paste:

```
#1a1b1d,#131415,#e9a825,#f2e5c7,#282828,#f2e5c7,#a5d6a7,#ff5f52,#1a1b1d,#bdae93
```
